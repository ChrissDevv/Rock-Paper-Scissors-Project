# Rock Paper Scissors

A simple browser-based Rock Paper Scissors game built with HTML and JavaScript.

## How to Play

- Click **Rock**, **Paper**, or **Scissors** to play a round against the computer
- The first to reach **5 points** wins the game
- The round result and running score are displayed on the page after each click

## Features

- Randomized computer choice each round
- Live score tracking
- Buttons automatically disable when the game is over

## Technologies Used

- HTML
- JavaScript (DOM manipulation)

## What I Learned

- Manipulating the DOM with `querySelector` and `textContent`
- Adding event listeners to buttons
- Managing game state with variables scoped outside of functions
- Refactoring console-based code into a browser UI
